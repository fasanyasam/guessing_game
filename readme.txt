# guessing_game

A Guessing Game 
This is a modified game which come with 3 levels 
    User is able to guess numbers against a number set by the program
    There are 3 levels, easy, medium and hard
    At easy, Users get a chance to guess the number between 1 - 10, and the user gets 6 guesses
    Medium, the users is required to guess the number between 1 - 20, and the user gets 4 guesses
    Hard, users are required to guess between 1 - 50, and they only get 3 guesses
    User are be able to set the level they want to play
    Users can see how many guesses they have left after each guess
    If the user guesses right, user is told "You got it right!"
    If the user guesses wrong, user is told "That was wrong"
    If the user uses all available guessing power and still unable to guess right, the user is told "Game over!"
